# String Notes

## Input

```cpp
string s;
cin >> s;           // one word
getline(cin, s);    // full line (watch leftover '\n' after cin >>)
```

## Useful

```cpp
s.size();           // or s.length()
s[i];               // char at i
reverse(s.begin(), s.end());  // #include <algorithm>
```
